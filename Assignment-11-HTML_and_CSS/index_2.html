Summary: Latitude vs. X

The purpose of this project was to visualize and analyze how different aspects of weather change with varying points of latitude. The different factors of weather that were chosen included Max Temperature, Humidity, Cloudiness, and Wind Speed.

Over 500 geographical coordinates were randomly generated using Python. The OpenWeather API was used to pull data for these random geographical coordinates, and the data was then compiled using Python.

The Matplotlib Python library was used to create visualizations. This site provides the source data and visualizations created as a part of the analysis, as well as explanations and descriptions of any trend and correlations witnessed.

Use the panel on the right to navigate between the different plots, or use the navigation bar at the top to view the plots and the source data. Use the button at the top left to navigate back to this homepage.
